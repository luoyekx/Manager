# Typecho 插件与主题管理器

在后台直接上传安装、卸载、备份、批量管理插件和主题，告别 FTP。

## 声明：本插件由 AI 开发，功能可用，但不保证没有 Bug，也不保证好用。使用前请自行备份重要数据，作者不承担因使用本插件造成的任何损失。

## 功能

- **在线上传安装** — 上传 ZIP 包直接安装插件或主题，支持拖拽、多文件同时上传
- **卸载** — 直接卸载插件或主题（自动禁用后再删除）
- **备份与下载** — 一键备份插件或主题为 ZIP，支持下载到本地
- **批量操作** — 全选/搜索/批量启用/批量禁用/批量备份/批量卸载
- **安全保护** — 不能卸载/禁用自身，不能卸载系统内置主题和当前使用的主题
- **备份管理** — 浏览、下载、删除备份文件，支持全部删除，过期自动清理
- **配置项** — 备份过期时间、上传大小限制、详细信息列显示

## 环境要求

- Typecho 1.2.0+
- PHP 7.4+
- PHP ZipArchive 扩展

## 安装

1. 下载最新版 [Manager.zip](https://github.com/luoyekx/Manager/releases)
2. 在 Typecho 后台「控制台 → 插件 → 上传插件」上传 ZIP，或手动解压到 `usr/plugins/Manager/`
3. 启用插件「Manager」
4. 进入「控制台 → 插件与主题管理」即可使用

## 使用说明

### 插件管理

- 点击上传区域或拖拽 ZIP 文件，可同时选择多个文件
- 单个插件：备份、卸载（需先禁用）
- 批量操作：勾选后点击对应按钮（启用/禁用/备份/卸载）
- 搜索框输入关键词实时过滤

### 主题管理

- 上传安装、备份、卸载主题
- 当前使用中的主题和系统内置主题受保护，不可卸载

### 备份管理

- 备份文件存储在 `usr/backups/`，目录受 `.htaccess` 保护
- 点击「刷新列表」加载所有备份文件
- 支持下载和删除单个备份，或全部删除
- 超过配置过期时间的备份文件会自动清理

## 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 备份过期时间 | 3600 秒 | 超过此时间未下载的备份将自动清理 |
| 最大上传大小 | 0（不限） | 超过此大小的 ZIP 将被拒绝，0 表示仅受 php.ini 限制 |
| 显示详细信息列 | 显示 | 在列表中显示版本号和作者信息 |

## 常见问题

**Q: 上传 ZIP 后提示"不是有效的 ZIP 文件"？**

A: 确保文件是标准的 ZIP 格式，Mac 用户请勿使用系统自带的压缩工具（会包含 `__MACOSX` 目录），推荐使用命令行 `zip -r` 打包。

**Q: 无法卸载插件？**

A: 请先在插件管理页面禁用该插件，再回到管理器卸载。

**Q: 上传的插件没有出现在插件列表中？**

A: 上传后需要到「插件管理」页面手动启用。
